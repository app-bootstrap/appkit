## Installment

``` bash
$ npm i app-debug -g
```

## Usage

### openurl

``` bash
$ app-debug openurl https://www.google.com
```
### gps

``` bash
$ app-debug gps 121.49612 31.24010
```

### install

``` bash
$ app-debug install xxx.apk or app
```

### start

``` bash
$ app-debug start processid
```
